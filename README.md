Project Overview :

AI ATS Resume Analyzer is a Generative AI-based web application that evaluates resumes against job descriptions and provides an ATS (Applicant Tracking System) compatibility score. The system helps candidates optimize their resumes to increase chances of passing automated hiring filters.
Key Features:

Upload Resume (PDF/DOC)
Job Description Matching
ATS Score Calculation
Skill Gap Analysis
Keyword Suggestions
Resume Improvement Recommendations
AI-powered feedback using LLM How It Works
User uploads resume
User provides job description
System extracts text from resume
AI model compares resume with job description
Generates:

ATS Score
Missing Keywords
Improvement Suggestions
